# What is

A form field is a components made to be used with form component. At this page we will list some things that all form field must have.

## List

- [ ] props interface consistent;
- [ ] css variables interface consistent;
- [ ] erro validation;
- [ ] integrate with form context and allow be used alone;
- [ ] have stories for:
  - custom css variables
  - values changed programmatically
  - list the css variables, with a description and the default value (can have examples too, like the theme stories)
- [ ] the interface to set value must allow svelte "bind" directive, and be similar to the svelte native way for the component that you are building
