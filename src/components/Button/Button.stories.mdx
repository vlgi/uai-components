# Button Custom

###### These are the component's CSS variables.

| CSS Property                    | Descripiton                                                            |
| ------------------------------- | ---------------------------------------------------------------------- |
| --szot-text-transform           | change text transformation mode                                        |
| --szot-opacity-hover            | change button opacity when hover                                       |
| --szot-effect-color-after-click | effect color when clicking the button                                  |
| --szot-color                    | Sets a custom text color                                               |
| --szot-background-color         | Set custom background color                                            |
| --szot-border                   | Set the border and its color                                           |
| --szot-border-radius            | change the radius of the button                                        |
| --szot-font-size                | change font size                                                       |
| --szot-padding                  | change button padding                                                  |
| --szot-margin-icon              | change the distance of the icon with the button (small, medium, large) |
