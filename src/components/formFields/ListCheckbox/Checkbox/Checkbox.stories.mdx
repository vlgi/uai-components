# Checkbox Custom

## Gerais

| CSS Property                 | Descripiton                | Default          |
| ---------------------------- | -------------------------- | ---------------- |
| --szot-checkbox-border-color | Set component border color | --theme-dark-txt |
| --szot-checkbox-border-size  | sets the border thickness  | 0.15em           |

## Custom variables for checkbox button

| CSS Property                | Descripiton               | Default          |
| --------------------------- | ------------------------- | ---------------- |
| --szot-checkbox-margin      | Set checkbox margin       | 0.3125rem        |
| --szot-checkbox-size        | Set checkbox button size  | 1rem             |
| --szot-checkbox-color       | Set checkbox button color | --theme-dark-txt |
| --szot-checkbox-label-color | Set checkbox label color  | --theme-dark-txt |

<p>
  This component returns the value (default: True) or False, if not checked.
</p>
