# Checkbox Custom

## Gerais

| CSS Property        | Descripiton                |
| ------------------- | -------------------------- |
| --szot-border-color | Set component border color |

## Custom variables for checkbox button

| CSS Property           | Descripiton               |
| ---------------------- | ------------------------- |
| --szot-checkbox-margin | Set checkbox margin       |
| --szot-checkbox-size   | Set checkbox button size  |
| --szot-checkbox-color  | Set checkbox button color |
| --szot-checkbox-label  | Set checkbox label color  |

<p>
  It is possible to create multiple checkbox button options using only this
  component, changing the names and IDs properly, but it's not recomended. You
  might rather try the "List Checkbox" component if you need to use many
  checkbox buttons.
</p>
