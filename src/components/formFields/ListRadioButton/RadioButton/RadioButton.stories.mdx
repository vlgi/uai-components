# Radio Button Custom

## Custom variables for radio button

| CSS Property             | Descripiton            | Default         |
| ------------------------ | ---------------------- | --------------- |
| --szot-radio-margin      | Set radio margin       | 0               |
| --szot-radio-size        | Set radio button size  | 1rem            |
| --szot-radio-color       | Set radio button color | --szot-dark-txt |
| --szot-radio-label-color | Set radio label color  | --szot-dark-txt |

<p>
  It is possible to create multiple radio button options using only this
  component, changing the names and IDs properly, but it's not recomended. You
  might rather try the "List Radio Button" component if you need to use many
  radio buttons.
</p>
