# Custom variables for select

You can set the following css variables for the select

| Variable                  | Default                 | Description |
|---------------------------|-------------------------|-------------|
| --szot-background-color   | white                   | This must be the same as the background color behind the input. |
| --szot-border-radius      | --szot-small-shape      |             |
| --szot-padding-vertical   | --szot-fields-padding   |             |
| --szot-padding-horizontal | --szot-fields-padding   |             |
| --szot-max-width          | --szot-fields-max-width |             |
