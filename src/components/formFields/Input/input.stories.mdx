# Input Custom

## Gerais

| CSS Property               | Descripiton                 |
| -------------------------- | --------------------------- |
| --szot-input-margin-bottom | Set component bottom margin |
| --szot-input-max-width     | Set component max width     |
| --szot-input-height        | Set component height        |

## On the input

| CSS Property               | Descripiton                        |
| -------------------------- | ---------------------------------- |
| --szot-input-top           | Set input position on top          |
| --szot-input-left          | Set the input position to the left |
| --szot-input-padding       | Set simple input padding           |
| --szot-input-color         | Set input text color               |
| --szot-input-border-color  | Set label focus position on top    |
| --szot-input-border-radius | set the border radius              |

## On the label

| CSS Property                  | Descripiton                                                                                           |
| ----------------------------- | ----------------------------------------------------------------------------------------------------- | --- |
| --szot-input-label-left       | Set the label position to the left                                                                    |
| --szot-input-label-padding    | Set the label padding                                                                                 |
| --szot-input-label-color      | Set label text color                                                                                  |     |
| --szot-input-background-color | Sets the background color of the label to hide the outline border. (Set according to page background) |     |

## When it's in focus

<p>
  {" "}
  These modifications can be used to change the color and position of input
  elements when it is in focus. If the component is not in focus but contains
  some text, only the colors will return to the default
</p>

| CSS Property                    | Descripiton                                 |
| ------------------------------- | ------------------------------------------- |
| --szot-input-label-focus-left   | Set the label position to the left          |
| --szot-input-focus-weigth       | Sets the font weight when input is in focus |
| --szot-input-focus-font-size    | Sets the font size when input is in focus   |
| --szot-input-focus-color        | Sets the font color when input is in focus  |
| --szot-input-border-color-focus | Set color to border when input is in focus  |

## With icons

| CSS Property             | Descripiton                                   |
| ------------------------ | --------------------------------------------- |
| --szot-input-icon-margin | Set the icon margin according to its position |
| --szot-input-icon-color  | Set the input color                           |

## Messages

| CSS Property                      | Descripiton                                                     |
| --------------------------------- | --------------------------------------------------------------- |
| --szot-input-message-font-size    | Set the font size of messages                                   |
| --szot-input-message-bottom       | Set messages position down                                      |
| --szot-input-message-left         | Set messages position to left                                   |
| --szot-input-message-bottom-focus | Set error text position to low when helper text is also showing |

<p>
  In addition to these variables, this component also uses the theme variables
  for, font-family, sizes, small-shape, small-border, the 4 main theme colors
  and the colors for error and information.
</p>
