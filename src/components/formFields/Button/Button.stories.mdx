# Button Custom

###### These are the component's CSS variables.

| CSS Property                           | Descripiton                                                            | Default                     |
| -------------------------------------- | ---------------------------------------------------------------------- | --------------------------- |
| --szot-button-text-transform           | change text transformation mode                                        | capitalize                  |
| --szot-button-opacity-hover            | change button opacity when hover                                       | 60%                         |
| --szot-button-effect-color-after-click | effect color when clicking the button                                  | rgba(184, 182, 182, 0.2)    |
| --szot-button-color                    | Sets a custom text color                                               | --theme-txt-on-dark-surface |
| --szot-button-background-color         | Set custom background color                                            | --theme-dark-surface        |
| --szot-button-border                   | Set the border and its color                                           | none (filled)               |
| --szot-button-border-radius            | change the radius of the button                                        | --theme-small-shape         |
| --szot-button-font-size                | change font size                                                       | --theme-fields-font-size    |
| --szot-button-padding                  | change button padding                                                  | 0.5625rem 3.4688rem(medium) |
| --szot-button-margin-icon              | change the distance of the icon with the button (small, medium, large) | -0.1rem 1rem                |
| --szot-button-max-width                | change the maximum width of the button                                 | --theme-fields-max-width    |
