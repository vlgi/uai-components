# Button Custom

###### These are the component's CSS variables.

| CSS Property                           | Descripiton                                                            |
| -------------------------------------- | ---------------------------------------------------------------------- |
| --szot-button-text-transform           | change text transformation mode                                        |
| --szot-button-opacity-hover            | change button opacity when hover                                       |
| --szot-button-effect-color-after-click | effect color when clicking the button                                  |
| --szot-button-color                    | Sets a custom text color                                               |
| --szot-button-background-color         | Set custom background color                                            |
| --szot-button-border                   | Set the border and its color                                           |
| --szot-button-border-radius            | change the radius of the button                                        |
| --szot-button-font-size                | change font size                                                       |
| --szot-button-padding                  | change button padding                                                  |
| --szot-button-margin-icon              | change the distance of the icon with the button (small, medium, large) |
| --szot-button-max-width                | change the maximum width of the button                                 |
