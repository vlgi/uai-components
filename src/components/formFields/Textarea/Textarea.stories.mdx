# Textarea Custom

## Gerais

| CSS Property                  | Descripiton                 | Default |
| ----------------------------- | --------------------------- | ------- |
| --szot-textarea-margin-top    | Set component top margin    | 0.5rem  |
| --szot-textarea-margin-bottom | Set component bottom margin | 1.5rem  |

## On the textarea

| CSS Property                      | Descripiton                                         | Default                   |
| --------------------------------- | --------------------------------------------------- | ------------------------- |
| --szot-textarea-max-width         | Sets the max-width of the textarea                  | 12rem                     |
| --szot-textarea-max-height        | sets the maximum height of the textarea             | undefined                 |
| --szot-textarea-font-size         | Set the font size of textarea                       | inherit                   |
| --szot-textarea-placeholder-color | Sets placeholder color                              | --szot-dark-txt           |
| --szot-textarea-color-text        | Set textarea text color (does not support gradient) | --szot-dark-inserted-text |
| --szot-textarea-border-color      | Sets the component's border color                   | --szot-dark-txt           |
| --szot-textarea-padding           | Sets a custom padding                               | --szot-fields-padding     |
| --szot-textarea-border            | Sets a custom border                                | --szot-small-border       |

## On the label

| CSS Property                      | Descripiton                                                                                           |
| --------------------------------- | ----------------------------------------------------------------------------------------------------- | --- |
| --szot-textarea-label-font-weight | Sets the weight of the label                                                                          |     |
| --szot-textarea-label-font-size   | Set the font size of the label                                                                        |
| --szot-textarea-label-padding     | Set the label padding                                                                                 |
| --szot-textarea-label-color       | Sets the color of the label                                                                           |
| --szot-textarea-background-color  | Sets the background color of the label to hide the outline border. (Set according to page background) |

## Messages

| CSS Property                               | Descripiton                                                     | Default |
| ------------------------------------------ | --------------------------------------------------------------- | ------- |
| --szot-textarea-message-bottom             | Set message text position to the bottom                         | -1.2rem |
| --szot-textarea-message-left               | Set message position to left                                    | 1rem    |
| --szot-textarea-message-error-bottom-focus | Set error text position to low when helper text is also showing | -2rem   |

<p>
  In addition to these variables, this component also uses the szot variables
  for, font-family, sizes, large-shape, small-border, the 4 main szot colors and
  the colors for error and information.
</p>

<p>
  The validation function to be passed to the component must be in the following
  format: The test function receives a "value" and uses this "value" to
  validate, the return must be "true" when valid, "false" when invalid (in this
  in case it is necessary to insert the "errorMsg " in the component) or the
  error message in string format.
</p>
