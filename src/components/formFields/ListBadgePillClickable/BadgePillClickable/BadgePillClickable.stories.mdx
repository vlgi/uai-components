# BadgePillClickable Custom

## Gerais

| CSS Property                            | Descripiton                        | Default                          |
| --------------------------------------- | ---------------------------------- | ---------------------------------|
| --szot-badge-pill-margin                | Set component margin               | 0.3125rem                        |
| --szot-badge-pill-min-height            | Set min height of the component    | 1.875rem                         |
| --szot-badge-pill-height                | Set the height of the component    | auto                             |
| --szot-badge-pill-border-size           | Set the border thickness           | 0.15rem                          |
| --szot-badge-pill-border-radius         | Set the border radius              | 1rem                             |
| --szot-badge-pill-border-color          | Set the border color               | --szot-dark-txt                  |
| --szot-badge-pill-label-color           | Set the label color                | --szot-dark-txt                  |
| --szot-badge-pill-icon-color            | Set the icon color                 | --szot-dark-txt                  |
| --szot-badge-pill-selected-border-color | Set the border color when selected | --szot-badge-pill-border-color   |
| --szot-badge-pill-selected-label-color  | Set the label color when selected  | --szot-badge-pill-label-color    |
| --szot-badge-pill-selected-icon-color   | Set the icon color when selected   | --szot-badge-pill-icon-color     |
| --szot-badge-pill-background-color      | Set the badge background color     | green                            |
| --szot-badge-pill-font-size             | Set the label font size            | --szot-fields-font-size          |
| --szot-badge-pill-max-width             | Set the badge max width            | auto                             |

<p>
  This component returns the value (default: True) or False, if not checked.
  If one color variable is set but the selected color variables isn't, then the color of badge pill
  selected will be the color when the badge is not selected.
</p>
