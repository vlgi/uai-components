# The menu

The menu is defined as fixed, and because of that the Docs page doens't work very well.

When using this menu you can set you own rule to says when use the desktop and mobile mode.

Be careful with the button and the menu position to not be behind another element.

# Customize Side Menu

You can customize the menu style using the following css variables.

## Sizes

<img src={menuSizeImg} alt="Menu size img" />

| Variable                   | Default |
| -------------------------- | ------- |
| --szot-nav-width           | 4.5rem  |
| --szot-nav-expanded-width  | 300px   |
| --szot-nav-logo-height     | 5rem    |
| --szot-nav-text-size       | 1rem    |
| --szot-nav-icon-size       | 1.5rem  |
| --szot-nav-item-max-height | 50px    |

## Paddings

<img src={menuPaddingImg} alt="Menu padding img" />

| Variable                    | Default     |
| --------------------------- | ----------- |
| --szot-nav-padding          | 0.4rem      |
| --szot-nav-logo-padding     | .6rem .3rem |
| --szot-nav-external-padding | 0px         |

**
NOTE: to ajust the logo image,
you must ajust the paddings and height to ajust the centralization.
**

## Colors

| Variable                              | Default                       | Description                                                              |
| ------------------------------------- | ----------------------------- | ------------------------------------------------------------------------ |
| --szot-nav-background-color           | #f0f0f0                       | The bg color of the menu, and button animation.                          |
| --szot-nav-txt-color                  | --szot-dark-txt               | The color of the menu itens and icons                                    |
| --szot-nav-items-hover-color          | --szot-light-surface          | The bg color of the menu itens                                           |
| --szot-nav-items-active-background    | --szot-dark-surface           | The bg color of the current active item                                  |
| --szot-nav-items-active-txt-color     | --szot-txt-on-dark-surface    | The text color of the current active item                                |
| --szot-nav-sub-items-active-highlight | --szot-items-active-txt-color | The underline color of the current active subitem, inside the group item |

## Other variables

| Variable                             | Default                                                   | Description                                                                               |
| ------------------------------------ | --------------------------------------------------------- | ----------------------------------------------------------------------------------------- |
| --szot-items-active-border           | none                                                      | You can set the border of the current active item instead of setting the background color |
| --szot-nav-animation-speed           | 350ms                                                     | Change the expand animation speed                                                         |
| --szot-nav-z-index                   | 9999                                                      | Set nav bar z-index                                                                       |
| --szot-nav-shadow                    | --szot-high-shadow                                        | Set the box shadow                                                                        |
| --szot-nav-border-radius             | 0 var(--szot-medium-shape) var(--szot-medium-shape) 0     | Set the nav border radius                                                                 |
| --szot-nav-items-border-radius       | 0px var(--szot-small-shape) var(--szot-small-shape) 0px   | Set the border radius of nav's items                                                      |
| --szot-nav-group-hover-border-radius | 0px var(--szot-medium-shape) var(--szot-medium-shape) 0px | Set the radius of the group item on hover                                                 |
