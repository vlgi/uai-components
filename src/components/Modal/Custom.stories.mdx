# Custom variables for Modal

You can set the following css variables for the modal

| Variable                  | Default                      | Description |
|---------------------------|------------------------------|-------------|
| --szot-width              | fit-content                  |             |
| --szot-height             | fit-content                  |             |
| --szot-max-height         | calc(100vh - 1.2rem)         |             |
| --szot-max-width          | calc(100vw - 1.2rem)         |             |
| --szot-overlay-color      | ccc                          |             |
| --szot-overlay-blur       | 2px                          |             |
| --szot-z-index            | 9999                         |             |
| --szot-close-bg-color     | --szot-light-surface         |             |
| --szot-close-txt-color    | --szot-txt-on-light-surface  |             |
| --szot-bg-color           | white                        |             |
