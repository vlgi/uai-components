# Textarea Custom

##### Gerais

<p>
  {" "}
  to customize the text colors of the textarea, label and borders it is
  necessary that the "textareaStyle" is "Custom"{" "}
</p>

| CSS Property         | Descripiton                 |
| -------------------- | --------------------------- |
| --szot-margin-bottom | Set component bottom margin |

##### On the textarea

| CSS Property                 | Descripiton                                |
| ---------------------------- | ------------------------------------------ |
| --szot-width                 | Sets the width of the textarea             |
| --szot-resize                | Sets whether an element is user resizable. |
| --szot-placeholder-font-size | Sets the placeholder font size             |
| --szot-placeholder-color     | Sets placeholder color                     |
| --szot-textarea-color        | Set textarea text color                    |
| --szot-border-color          | Sets the component's border color          |

##### On the label

| CSS Property             | Descripiton                                                                                           |
| ------------------------ | ----------------------------------------------------------------------------------------------------- | --- |
| --szot-label-line-height | Sets the height of the label line                                                                     |
| --szot-label-font-weight | Sets the weight of the label                                                                          |     |
| --szot-label-font-size   | Set the font size of the label                                                                        |
| --szot-label-padding     | Set the label padding                                                                                 |
| --szot-label-color       | Sets the color of the label                                                                           |
| --szot-background-color  | Sets the background color of the label to hide the outline border. (Set according to page background) |

##### Messages

| CSS Property                  | Descripiton                                                     |
| ----------------------------- | --------------------------------------------------------------- |
| --szot-message-font-size      | Set the font size of messages                                   |
| --szot-texthelp-bottom        | Set helper text position down                                   |
| --szot-texthelp-left          | Set helper text position to left                                |
| --szot-texterror-bottom       | Set the error text position to the bottom                       |
| --szot-texterror-left         | Set error text position to left                                 |
| --szot-texterror-bottom-focus | Set error text position to low when helper text is also showing |

<p>
  In addition to these variables, this component also uses the theme variables
  for, font-family, sizes, large-shape, small-border, the 4 main theme colors
  and the colors for error and information.
</p>

<p>
  The validation function to be passed to the component must be in the following
  format: The test function receives a "value" and uses this "value" to
  validate, the return must be "true" when valid, "false" when invalid (in this
  in case it is necessary to insert the "errorMsg " in the component) or the
  error message in string format.
</p>

##### below is an example of the modified component using some of the variables

<p>
  Here a function was applied to validate and if you write "show an error" the
  component will be invalid.
</p>

export const Template = (args) => ({
  Component: Textarea,
  props: args,
});

export let validteste = (value) => {
  if (value === "mostre um erro") {
    return "Aqui está o erro.";
  }
  return true;
};
