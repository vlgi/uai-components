# Dialog

This component is just a wrapper to add some preset with buttons and dialog logic to the modal.

We use our `Modal` and `Button` components, so you can customize this setting the modal an button
css variables.

For the content, header, and footer, you can change every thing using the slots.

| CSS Property                     | Descripiton                                   | Default                    |
| -------------------------------- | --------------------------------------------- | -------------------------- |
| --szot-dialog-text-color         | change the color of text inside dialog        | --szot-txt-on-signal-color |
| --szot-dialog-text-color-header  | change the color of text inside dialog header | --szot-txt-on-signal-color |
| --szot-dialog-text-weight        | change weight of text inside dialog           | 400                        |
| --szot-dialog-text-weight-header | change weight of text inside dialog header    | 700                        |
| --szot-dialog-text-size          | change size of text inside dialog             | 0.8125rem                  |
| --szot-dialog-text-size-header   | change size of text inside dialog header      | 0.875rem                   |

You can also use custom variables from [Modal](?path=/story/components-modal-custom--page) component to change Dialog, once the Dialog is a Modal.

If you want to change only the dialog properties, you need to reference globally the `dialog-wrapper-container`
class and change css properties of modal inside this class.

# Buttons Style

For change style of buttons of dialog separately, you need to reference globally the classes of every button
container and change the css custom properties of [Buttons](?path=/story/components-formfields-button-custom--page) inside every container.

| Button         | Container Class          |
| -------------- | ------------------------ |
| cancel button  | cancel-button-container  |
| confirm button | confirm-button-container |
| other button   | others-button-container  |
