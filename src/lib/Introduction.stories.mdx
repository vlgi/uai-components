# Szot UI

## Introduction
This package use css variables to set theme, you can see how set them [here](?path=/story/theme-introduction--page).

The components docs is inside the `Components` section.

You can see the component example inside the `Canvas`, swithing in the tab menu at the top.

You can see the props and all examples inside the `Docs`, swithing in the tab menu at the top.

Every component that can be customized with custom css variables have a section `Custom`, inside the component folder.

**Note: This lib is made to be used only with `svelte`, is `experimental`, and may NOT follow the `semver`.**

## How to install

Install the package:

```bash
npm install szot-ui
```

Import the global style and icons, in your entry point or global css:

```scss
import "szot-ui/src/styles/global.scss";
```

**NOTE: make sure that you bundler can handle with scss, images and fonts, to import the styles and icons**

Use the components =)

```html
<script lang="ts">
  import {
    Input,
    Form,
  } from "szot-ui";

  export let values: Record<string, unknown>;
  export let isAllValid: boolean;

  function validateEmail(value: string): string|boolean {
    if (/[^@ \t\r\n]+@[^@ \t\r\n]+\.[^@ \t\r\n]+/.test(value)) return true;
    return "Type a valid e-mail.";
  }
</script>

<div class="my-form">
  <Form bind:values bind:isAllValid on:submit>
    <Input
      label="email"
      name="email"
      type="email"
      iconPosition="left"
      icon="mail"
      validationFn={validateEmail}
      required={true}
    />
    <Button type="submit">Submit</Button>
  </Form>
```
