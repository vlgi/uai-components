# Collapsable Css Variables

###### These are the component's CSS variables.

| CSS Property                      | Descripiton                                                                                         | Default                                                                                                                                  |
| --------------------------------- | --------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------- |
| --uai-collapsable-background      | set a custom background color for the whole collapsable                                             | --uai-primary-surface                                                                                                                    |
| --uai-collapsable-color           | set a custom text color for the whole collapsable (or only title when contentBackground is enabled) | --uai-txt-on-primary-surface                                                                                                             |
| --uai-collapsable-border-color    | set a custom border color around the collapsable                                                    | none                                                                                                                                     |
| --uai-collapsable-icon-color      | set a custom icon color                                                                             | --uai-txt-on-primary-surface                                                                                                             |
| --uai-collapsable-max-height      | set a max height for the content. The content will scroll if it is taller than what you set         | 40rem                                                                                                                                    |
| --uai-collapsable-icon-font-size  | set a size for the chevron icon                                                                     | 1.5rem                                                                                                                                   |
| --uai-collapsable-margin          | set a margin around the collapsable                                                                 | --uai-global-medium-padding                                                                                                              |
| --uai-collapsable-title-font-size | set the size of the collapsable title                                                               | 1rem                                                                                                                                     |
| --uai-collapsable-padding         | set a padding for the collapsable                                                                   | --uai-global-medium-padding                                                                                                              |
| --uai-collapsable-padding-header  | set a padding for the collapsable header                                                            | --uai-global-medium-padding                                                                                                              |
| --uai-collapsable-border-width    | set a width for the border around the collapsable                                                   | --uai-medium-border                                                                                                                      |
| --uai-collapsable-border-radius   | set a radius for the collapsable                                                                    | calc( calc( max(var(--icon-font-size), 1.1875rem) + calc(2 \* var(--theme-global-medium-padding)) ) / 2 )) // this is half of the height |
| --uai-collapsable-box-shadow      | set box shadow for the collapsable                                                                  | none                                                                                                                                     |
| --uai-collapsable-backdrop-filter | set backdrop filter for the collapsable                                                             | none                                                                                                                                     |

###### These only work when contentBackground is enabled

| CSS Property                            | Descripiton                                                                     | Default                     |
| --------------------------------------- | ------------------------------------------------------------------------------- | --------------------------- |
| --uai-collapsable-content-background    | set a custom content background color when contentBackground is enabled         | --uai-light-surface         |
| --uai-collapsable-content-color         | set a custom content color when contentBackground is enabled                    | --uai-txt-on-light-surface  |
| --uai-collapsable-content-border-color  | set a custom border color around the content when contentBackground is enabled  | none                        |
| --uai-collapsable-content-padding       | set a padding for the content when the contentBackground is enabled             | --uai-global-medium-padding |
| --uai-collapsable-content-border-width  | set a width for the border around the content when contentBackground is enabled | --uai-medium-border         |
| --uai-collapsable-content-border-radius | set a radius for the content when the contentBackground is enabled              | --uai-medium-shape          |
