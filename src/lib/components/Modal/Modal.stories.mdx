# CSS variables for Modal

You can set the following css variables for the modal

| Variable                    | Default                         |
| --------------------------- | ------------------------------- |
| --uai-modal-width           | fit-content                     |
| --uai-modal-height          | fit-content                     |
| --uai-modal-max-height      | calc(100vh - 1.2rem)            |
| --uai-modal-max-width       | calc(100vw - 1.2rem)            |
| --uai-modal-overlay-color   | ccc                             |
| --uai-modal-overlay-blur    | 2px                             |
| --uai-modal-z-index         | 9999                            |
| --uai-modal-bg-color        | white                           |
| --uai-modal-close-bg-color  | --uai-light-surface             |
| --uai-modal-close-txt-color | --uai-txt-on-light-surface      |
| --uai-modal-padding         | 0.6rem                          |
| --uai-modal-margin-content  | 0.6rem 0                        |
| --uai-modal-header-padding  | 0.9375rem 0.9375rem 0 0.9375rem |
| --uai-modal-footer-padding  | 0 0.9375rem 0.9375rem 0.9375rem |
| --uai-modal-content-padding | 0 0.9375rem 0.9375rem 0.9375rem |
