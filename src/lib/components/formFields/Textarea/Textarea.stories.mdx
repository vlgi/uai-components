# Textarea CSS Variables

## Gerais

| CSS Property                 | Descripiton                 | Default |
| ---------------------------- | --------------------------- | ------- |
| --uai-textarea-margin-top    | Set component top margin    | 0.5rem  |
| --uai-textarea-margin-bottom | Set component bottom margin | 1.5rem  |

## On the textarea

| CSS Property                       | Descripiton                                                    | Default                  |
| ---------------------------------- | -------------------------------------------------------------- | ------------------------ |
| --uai-textarea-max-width           | Sets the max-width of the textarea                             | 100%                     |
| --uai-textarea-max-height          | sets the maximum height of the textarea                        | undefined                |
| --uai-textarea-font-size           | Set the font size of textarea                                  | inherit                  |
| --uai-textarea-placeholder-color   | Sets placeholder color                                         | --uai-dark-txt           |
| --uai-textarea-color-text          | Set textarea text color (does not support gradient)            | --uai-dark-inserted-text |
| --uai-textarea-border-color-focus  | Sets the component's border color when the component is focus  | --uai-dark-txt           |
| --uai-textarea-border-color-filled | Sets the component's border color when the component is filled | --uai-dark-txt           |
| --uai-textarea-border-color        | Sets the component's border color                              | --uai-dark-txt           |
| --uai-textarea-padding             | Sets a custom padding                                          | --uai-fields-padding     |
| --uai-textarea-border              | Sets a custom border                                           | --uai-small-border       |

## On the label

| CSS Property                     | Descripiton                                                                                           |
| -------------------------------- | ----------------------------------------------------------------------------------------------------- | ----------- |
| --uai-textarea-label-font-weight | Sets the weight of the label                                                                          |             |
| --uai-textarea-label-font-size   | Set the font size of the label                                                                        |             |
| --uai-textarea-label-padding     | Set the label padding                                                                                 |             |
| --uai-textarea-label-color       | Sets the color of the label                                                                           |             |
| --uai-textarea-background-color  | Sets the background color of the label to hide the outline border. (Set according to page background) | transparent |

## Messages

| CSS Property                              | Descripiton                                                     | Default |
| ----------------------------------------- | --------------------------------------------------------------- | ------- |
| --uai-textarea-message-bottom             | Set message text position to the bottom                         | -1.2rem |
| --uai-textarea-message-left               | Set message position to left                                    | 1rem    |
| --uai-textarea-message-error-bottom-focus | Set error text position to low when helper text is also showing | -2rem   |
| --uai-textarea-error-height-hidden        | Set error text height when the error is hidden                  | 0       |

<p>
  In addition to these variables, this component also uses the UAI global variables for,
  font-family, sizes, large-shape, small-border, the 4 main UAI colors and the colors for error and
  information.
</p>

<p>
  The validation function to be passed to the component must be in the following format: The test
  function receives a "value" and uses this "value" to validate, the return must be "true" when
  valid, "false" when invalid (in this in case it is necessary to insert the "errorMsg " in the
  component) or the error message in string format.
</p>
