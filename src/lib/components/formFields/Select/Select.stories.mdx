# CSS Variables

You can set the following css variables for the select

| Variable                          | Default                      | Description                                                     |
| --------------------------------- | ---------------------------- | --------------------------------------------------------------- |
| --szot-select-background-color    | transparent                  | this must be the same as the background color behind the input. |
| --szot-select-border-radius       | --szot-select-small-shape    |                                                                 |
| --szot-select-padding-vertical    | --szot-select-fields-padding |                                                                 |
| --szot-select-padding-horizontal  | --szot-select-fields-padding |                                                                 |
| --szot-select-max-width           | 100%                         |                                                                 |
| --szot-select-label-color         | --szot-dark-txt              |                                                                 |
| --szot-select-border-color        | --szot-dark-txt              | color of the border of select                                   |
| --szot-select-margin-top          | 0.5rem                       |                                                                 |
| --szot-select-badge-color         | --theme-dark-txt             |                                                                 |
| --szot-select-badge-border-color  | --theme-dark-txt             |                                                                 |
| --szot-select-border-color-focus  | --theme-dark-txt             | color of the borders of the select when it is focus             |
| --szot-select-border-color-filled | --theme-dark-txt             | color of the borders of the select when it is filled            |
| --select-label-margin-right       | 1,3rem                       | margin on the right so the text is not under the icon           |
| --szot-select-border              | --theme-small-border         | width of the border                                             |
| --szot-select-text-color          | --theme-dark-txt             | color of the texts of the select                                |
| --szot-select-placeholder-color   | --theme-dark-txt             | color of the placeholder text                                   |

You can use badge variables to customize but the text and border color must be changed with the
specific variable described above.
