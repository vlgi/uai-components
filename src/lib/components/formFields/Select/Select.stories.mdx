# CSS Variables

You can set the following css variables for the select

| Variable                                  | Default                 | Description                                                     |
| ----------------------------------------- | ----------------------- | --------------------------------------------------------------- |
| --uai-select-background-color             | transparent             | this must be the same as the background color behind the input. |
| --uai-select-border-radius                | --uai-small-shape       |                                                                 |
| --uai-select-padding-vertical             | --uai-fields-padding    |                                                                 |
| --uai-select-padding-horizontal           | --uai-fields-padding    |                                                                 |
| --uai-select-max-width                    | 100%                    |                                                                 |
| --uai-select-label-color                  | --uai-dark-txt          |                                                                 |
| --uai-select-border-color                 | --uai-dark-txt          | color of the border of select                                   |
| --uai-select-margin-top                   | 0.5rem                  |                                                                 |
| --uai-select-badge-color                  | --uai-dark-txt          |                                                                 |
| --uai-select-badge-border-color           | --uai-dark-txt          |                                                                 |
| --uai-select-border-color-focus           | --uai-dark-txt          | color of the borders of the select when it is focus             |
| --uai-select-border-color-filled          | --uai-dark-txt          | color of the borders of the select when it is filled            |
| --select-label-margin-right               | 1,3rem                  | margin on the right so the text is not under the icon           |
| --uai-select-border                       | --uai-small-border      | width of the border                                             |
| --uai-select-text-color                   | --uai-dark-txt          | color of the texts of the select                                |
| --uai-select-placeholder-color            | --uai-dark-txt          | color of the placeholder text                                   |
| --uai-select-floating-dropdown-bg-color   | white                   |                                                                 |
| --uai-select-floating-dropdown-text-color | --uai-select-text-color |                                                                 |
| --uai-select-error-height-hidden          |                         | height of the error text below select when the text is hidden   |
| --uai-select-dropdown-max-height          | 10rem                   |                                                                 |

You can use badge variables to customize but the text and border color must be changed with the
specific variable described above.
