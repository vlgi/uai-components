# Button Css Variables

| CSS Property                           | Descripiton                                                            | Default                     |
| -------------------------------------- | ---------------------------------------------------------------------- | --------------------------- |
| --szot-button-text-transform           | change text transformation mode                                        | capitalize                  |
| --szot-button-opacity-hover            | change button opacity when hover                                       | 60%                         |
| --szot-button-effect-color-after-click | effect color when clicking the button                                  | rgba(184, 182, 182, 0.2)    |
| --szot-button-color                    | Set a custom text color                                                | --szot-txt-on-dark-surface  |
| --szot-button-icon-color               | Set a custom icon color                                                | --szot-button-color         |
| --szot-button-background-color         | Set custom background color                                            | --szot-dark-surface         |
| --szot-button-border                   | Set the border size                                                    | none (filled)               |
| --szot-button-border-color             | set border color                                                       | #333                        |
| --szot-button-border-radius            | change the radius of the button                                        | --szot-small-shape          |
| --szot-button-font-size                | change font size                                                       | --szot-fields-font-size     |
| --szot-button-padding                  | change button padding                                                  | 0.5625rem 3.4688rem(medium) |
| --szot-button-margin-icon              | change the distance of the icon with the button (small, medium, large) | -0.1rem 1rem                |
| --szot-button-max-width                | change the maximum width of the button                                 | 100%                        |
| --szot-button-width                    | sets the width of the button                                           | 100%                        |
| --szot-button-white-space              | sets the white space css property of button text                       | nowrap                      |
| --szot-button-box-shadow               | sets the box shadow of the button                                      | none                        |
| --szot-button-backdrop-filter          | sets the backdrop filter of the button                                 | none                        |
