# List Checkbox Css Variables

## Gerais

| CSS Property                  | Descripiton                 | Default |
| ----------------------------- | --------------------------- | ------- |
| --szot-checkbox-border-radius | Set component border radius | 0.15em  |
| --szot-checkbox-border-size   | sets the border thickness   | 0.15em  |

## Custom variables for checkbox button

| CSS Property                | Descripiton                | Default         |
| --------------------------- | -------------------------- | --------------- |
| --szot-checkbox-margin      | Set checkbox margin        | 0.3125rem       |
| --szot-checkbox-size        | Set checkbox button size   | 1rem            |
| --szot-checkbox-color       | Set component border color | --szot-dark-txt |
| --szot-checkbox-label-color | Set checkbox label color   | --szot-dark-txt |

<p>This component returns the list of checked values.</p>
