# List Checkbox Css Variables

## Gerais

| CSS Property                 | Descripiton                 | Default |
| ---------------------------- | --------------------------- | ------- |
| --uai-checkbox-border-radius | Set component border radius | 0.15em  |
| --uai-checkbox-border-size   | sets the border thickness   | 0.15em  |

## Custom variables for checkbox button

| CSS Property               | Descripiton                | Default        |
| -------------------------- | -------------------------- | -------------- |
| --uai-checkbox-margin      | Set checkbox margin        | 0.3125rem      |
| --uai-checkbox-size        | Set checkbox button size   | 1rem           |
| --uai-checkbox-color       | Set component border color | --uai-dark-txt |
| --uai-checkbox-label-color | Set checkbox label color   | --uai-dark-txt |

## List Checkbox display positions

| CSS Property                        | Descripiton                                        | Default |
| ----------------------------------- | -------------------------------------------------- | ------- |
| --uai-list-checkbox-justify-content | Set justify content variable of checkbox container |         |
| --uai-list-checkbox-align-items     | Set align items variable of checkbox container     |         |

<p>This component returns the list of checked values.</p>
