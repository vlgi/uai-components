# Checkbox Css Variables

| CSS Property                     | Description                             | Default          |
| -------------------------------- | --------------------------------------- | ---------------- |
| --uai-checkbox-margin            | Set checkbox margin                     | 0.3125rem        |
| --uai-checkbox-padding           | Set checkbox padding                    | 0 0.375em        |
| --uai-checkbox-size              | Set checkbox button size                | 1rem             |
| --uai-checkbox-color             | Set component color                     | --uai-dark-txt   |
| --uai-checkbox-checked-color     | Set component color on checked          | --checkbox-color |
| --uai-checkbox-label-color       | Set checkbox label color                | --uai-dark-txt   |
| --uai-checkbox-border-radius     | Set component border radius             | 0.15em           |
| --uai-checkbox-border-size       | Set the border thickness                | 0.15em           |
| --uai-checkbox-label-margin-left | set distance between checkbox and label | 1.25rem          |

## Custom variables for checkbox button with "switch" styleType

| CSS Property                        | Description                             | Default          |
| ----------------------------------- | --------------------------------------- | ---------------- |
| --uai-checkbox-switch-background    | Set switch background                   | #cecece          |
| --uai-checkbox-switch-color         | Set switch button color                 | --theme-dark-txt |
| --uai-checkbox-switch-checked-color | Set switch color when checked           | --switch-color   |
| --uai-checkbox-switch-width         | Set switch button width                 | 3.25rem          |
| --uai-checkbox-switch-height        | Set switch button height                | 1.5rem           |
| --uai-checkbox-border-radius        | Set component border radius             | 2.125rem         |
| --uai-checkbox-label-margin-left    | set distance between checkbox and label | 1.25rem          |
