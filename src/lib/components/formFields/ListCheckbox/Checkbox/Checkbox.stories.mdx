# Checkbox Custom

<p>You can change the css variables globally, like:</p>

```css
:root {
  --szot-checkbox-border-radius: 0.5rem;
}
```

<p>
  But you can change globally the css variables of a specific style type by
  setting the values of the variables of the class .checkbox.style-type-
  <strong>styleType</strong>, where <strong>styleType</strong> is the name of the
  style type ("checkbox-input" , "switch" , "text-switch"). If you want to use this
  feature locally, you need to use the where css feature.
</p>

<p>
  Using <strong>:where()</strong> we can globally define the variables of each
  type without affecting the others that use the same variable. in case you need
  to read the documentation it is
  <a href="https://developer.mozilla.org/en-US/docs/Web/CSS/:where"> here.</a>
</p>

```css
:where(.checkbox.style-type-checkbox-input) {
  --szot-checkbox-border-radius: 0.5rem;
}

:where(.checkbox.style-type-switch) {
  --szot-checkbox-switch-width: 3rem;
}

:where(.checkbox.style-type-text-switch) {
  --szot-checkbox-switch-width: 5rem;
  --szot-checkbox-border-radius: 6rem;
}
```

## Custom variables for checkbox button

| CSS Property                      | Description                             | Default          |
| --------------------------------- | --------------------------------------- | ---------------- |
| --szot-checkbox-margin            | Set checkbox margin                     | 0.3125rem        |
| --szot-checkbox-padding           | Set checkbox padding                    | 0 0.375em        |
| --szot-checkbox-size              | Set checkbox button size                | 1rem             |
| --szot-checkbox-color             | Set component color                     | --szot-dark-txt  |
| --szot-checkbox-checked-color     | Set component color on checked          | --checkbox-color |
| --szot-checkbox-label-color       | Set checkbox label color                | --szot-dark-txt  |
| --szot-checkbox-border-radius     | Set component border radius             | 0.15em           |
| --szot-checkbox-border-size       | Set the border thickness                | 0.15em           |
| --szot-checkbox-label-margin-left | set distance between checkbox and label | 1.25rem          |

| CSS Property                         | Description                             | Default          |

| --szot-checkbox-switch-checked-color | Set switch color when checked           | --switch-color   |
| --szot-checkbox-switch-width         | Set switch button width                 | 3.25rem          |
| --szot-checkbox-switch-height        | Set switch button height                | 1.5rem           |
| --szot-checkbox-border-radius        | Set component border radius             | 2.125rem         |
| --szot-checkbox-label-margin-left    | set distance between checkbox and label | 1.25rem          |
