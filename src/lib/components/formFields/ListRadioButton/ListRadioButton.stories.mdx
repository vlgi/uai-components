# List Radio Button Css Variables

| CSS Property            | Descripiton            | Default        |
| ----------------------- | ---------------------- | -------------- |
| --uai-radio-margin      | Set radio margin       | 0              |
| --uai-radio-size        | Set radio button size  | 1rem           |
| --uai-radio-color       | Set radio button color | --uai-dark-txt |
| --uai-radio-label-color | Set radio label color  | --uai-dark-txt |

| --uai-list-radio-align-items     | Set align items variable of radio container     |         |
