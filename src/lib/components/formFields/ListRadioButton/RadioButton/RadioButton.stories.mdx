# Radio Button Css Variables

| CSS Property               | Description                                              | Default         |
| -------------------------- | ---------------------------------------------------------| --------------- |
| --szot-radio-margin        | Set radio margin                                         | 0               |
| --szot-radio-size          | Set radio button size                                    | 1rem            |
| --szot-radio-filled-gap    | Set distance between the border and the filled circle    | 0.5rem          |
| --szot-radio-gap-label     | Set distance between the label and the button            | 0               |
| --szot-radio-border        | set button border size                                   | 0.125em         |
| --szot-radio-checked-color | set the button color when the item is checked            | --radio-color   |
| --szot-radio-color         | Set radio button color                                   | --szot-dark-txt |
| --szot-radio-label-color   | Set radio label color                                    | --szot-dark-txt |

## Custom variables for checkbox button with "badge-pill" styleType

| CSS Property                                  | Description                                      | Default                  |
| --------------------------------------------- | -------------------------------------------------| ------------------------ |
| --szot-radio-badge-pill-min-height            | Set min height of the badge pill                 | 1.875rem                 |
| --szot-radio-badge-pill-height                | Set height of the badge pill                     | auto                     |
| --szot-radio-badge-pill-border-radius         | Set border radius of the badge pill              | 1rem                     |
| --szot-radio-badge-pill-background-color      | Set background color of the badge pill           | #aaa                     |
| --szot-radio-badge-pill-max-width             | Set max width of the badge pill                  | auto                     |
| --szot-radio-badge-pill-font-size             | Set font size of the badge pill label            | --theme-fields-font-size |
| --szot-radio-badge-pill-font-weight           | Set font weight of the badge pill label          | normal                   |
| --szot-radio-badge-pill-border-color          | Set border color of the badge pill               | --default-border-color   |
| --szot-radio-badge-pill-label-color           | Set label color of the badge pill                | --default-label-color    |
| --szot-radio-badge-pill-icon-color            | Set icon color of the badge pill                 | --default-icon-color     |
| --szot-radio-badge-pill-selected-border-color | Set border color of the badge pill when selected | --default-border-color   |
| --szot-radio-badge-pill-selected-label-color  | Set label color of the badge pill when selected  | --default-label-color    |
| --szot-radio-badge-pill-selected-icon-color   | Set icon color of the badge pill when selected   | --default-icon-color     |
