# Input Css Variables

## Gerais

| CSS Property              | Descripiton                 | Default |
| ------------------------- | --------------------------- | ------- |
| --uai-input-margin-top    | Set component top margin    | 0.5rem  |
| --uai-input-margin-bottom | Set component bottom margin | 1.5rem  |
| --uai-input-max-width     | Set component max width     | 100%    |

## On the input

| CSS Property                  | Descripiton                                      | Default                  |
| ----------------------------- | ------------------------------------------------ | ------------------------ |
| --uai-input-top               | Set input position on top                        | 0                        |
| --uai-input-left              | Set the input position to the left               | 0                        |
| --uai-input-padding           | Set simple input padding                         | --uai-fields-padding     |
| --uai-input-text-color        | Set input text color (does not support gradient) | --uai-dark-inserted-text |
| --uai-input-border            | Set border width                                 | --theme-small-border     |
| --uai-input-border-color      | Set label focus position on top                  | --uai-dark-txt           |
| --uai-input-border-radius     | set the border radius                            | --uai-small-shape        |
| --uai-input-placeholder-color | set the placeholder color                        | --uai-dark-txt           |

## On the label

| CSS Property                 | Descripiton                                                                                          | Default        |
| ---------------------------- | ---------------------------------------------------------------------------------------------------- | -------------- |
| --uai-input-label-left       | Set the label position to the left                                                                   | 1.2rem         |
| --uai-input-label-padding    | Set the label padding                                                                                | 0              |
| --uai-input-label-color      | Set label text color                                                                                 | --uai-dark-txt |
| --uai-input-background-color | Set the background color of the label to hide the outline border. (Set according to page background) | transparent    |

## When it's in focus

<p>
  {" "}
  These modifications can be used to change the color and position of input elements when it is in focus.
  If the component is not in focus but contains some text, only the colors will return to the default
</p>

| CSS Property                      | Descripiton                                              | Default                                   |
| --------------------------------- | -------------------------------------------------------- | ----------------------------------------- |
| --uai-input-label-focus-left      | Set the label position to the left                       | 0                                         |
| --uai-input-label-focus-color     | Set label color when input is in focus                   | --uai-input-label-color (--uai-dark-txt)  |
| --uai-input-label-not-focus-color | Set label color when input is not in focus but not empty | --uai-input-label-color (--uai-dark-txt)  |
| --uai-input-border-color-focus    | Set color to border when input is in focus               | --uai-input-border-color (--uai-dark-txt) |
| --uai-input-border-color-filled   | Set color to border when input is in filled              | --uai-input-border-color (--uai-dark-txt) |

## With icons

| CSS Property                  | Descripiton                     | Default                                                |
| ----------------------------- | ------------------------------- | ------------------------------------------------------ |
| --uai-input-icon-color        | Set the input icon color        | --uai-dark-txt                                         |
| --uai-input-icon-margin-right | Set the input icon margin right | 0 if icon in the left and 0.5 rem if icon in the right |
| --uai-input-icon-margin-left  | Set the input icon margin left  | 0 if icon in the right and 0.5 rem if icon in the left |

## Messages

| CSS Property                           | Descripiton                                                     | Default                      |
| -------------------------------------- | --------------------------------------------------------------- | ---------------------------- |
| --uai-input-message-top                | Set messages position top                                       | 37.19 + (--uai-input-border) |
| --uai-input-message-left               | Set messages position to left                                   | 1rem                         |
| --uai-input-message-error-bottom-focus | Set error text position to low when helper text is also showing | -2rem                        |
| --uai-input-error-height-hidden        | Set error text height when the error is hidden                  | 0                            |

<p>
  In addition to these variables, this component also uses the UAI global css variables for,
  font-family, sizes, small-shape, small-border, the 4 main UAI colors and the colors for error and
  information.
</p>
