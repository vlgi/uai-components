# Edit Label

| CSS Property                          | Descripiton         | Default     |
| ------------------------------------- | ------------------- | ----------- |
| --szot-edit-label-width               | modal width         | 25rem       |
| --szot-edit-label-height              | modal height        | 22.5rem     |
| --szot-edit-label-input-margin-top    | input margin top    | 0           |
| --szot-edit-label-input-margin-bottom | input margin bottom | 0           |
| --szot-edit-label-input-border        | input border size   | none        |
| --szot-edit-label-input-border-color  | input border color  | transparent |
| --szot-edit-label-border-radius       | input border radius | 1.5rem      |

## Notes

The others input variables can be customized here, as well as the Color Picker variables.
