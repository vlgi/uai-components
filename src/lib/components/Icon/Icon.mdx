# Icon

You can get the list of icon names in the [iconify page](https://icon-sets.iconify.design/)

To add icons in the icon bundle to render offline, import the function addIcons from helper folder
or from the uai-components (`import {addIcons} from "uai-components"`) if you are using it in your
own project. The parameter of the function addIcons is a list of IconifyJSON objects

To get the IconifyJSON objects to use in the addIcons function, use the api query from iconify to
get the objects.

The api query is "https://api.iconify.design/{prefix}.json?icons={icons}", where icons is the list
of icons separated by comma.

Ex: https://api.iconify.design/mdi.json?icons=account-box,account-cash,account

## Usage

```tsx
<script lang="ts">
  import { Icon } from "uai-components";
</script>

<Icon
  iconName={"ic:baseline-apple"}
  --uai-icon-color="red"
  --uai-icon-font-size="80px"
/>
```
