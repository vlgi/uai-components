# CSS variables

You can set the following css variables for the icon

| Variable             | Default          | Description               |
| -------------------- | ---------------- | ------------------------- |
| --uai-icon-color     | --theme-dark-txt | Set the color of the icon |
| --uai-icon-font-size | 1.25rem          | Set the size of the icon  |
