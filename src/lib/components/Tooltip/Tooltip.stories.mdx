# CSS variables for Tooltip

You can set the following css variables for the Tooltip

| Variable                       | Default                         |
| ------------------------------ | ------------------------------- |
| --szot-tooltip-txt-color       | #ffffff                         |
| --szot-tooltip-txt-font-size   | 1rem                            |
| --szot-tooltip-txt-font-weight | 400                             |
