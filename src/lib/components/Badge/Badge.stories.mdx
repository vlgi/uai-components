# Badge Css Variables

| CSS Property                 | Default                       | Descripiton                             |
| ---------------------------- | ----------------------------- | --------------------------------------- |
| --uai-badge-color            | --uai-txt-on-primary-surface  | Defined following the prop "badgeStyle" |
| --uai-badge-background-color | --uai-dark-surface            | Defined following the prop "badgeStyle" |
| --uai-badge-border           | --uai-small-border            | Defined following the prop "badgeStyle" |
| --uai-badge-font-size        | .9rem                         | set the badge font size                 |
| --uai-badge-border-color     | #333                          | set the badge border color              |
| --uai-badge-padding          | 0 var(--theme-fields-padding) |                                         |

<p>You can use a gradient or image for the background and text color.</p>
