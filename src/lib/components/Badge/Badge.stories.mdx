# Badge Custom

| CSS Property                  | Default                       | Descripiton                             |
| ----------------------------- | ----------------------------- | --------------------------------------- |
| --szot-badge-color            | --szot-txt-on-primary-surface | Defined following the prop "badgeStyle" |
| --szot-badge-background-color | --szot-dark-surface           | Defined following the prop "badgeStyle" |
| --szot-badge-border           | --szot-small-border           | Defined following the prop "badgeStyle" |
| --szot-badge-font-size        | .9rem                         | set the badge font size                 |
| --szot-badge-border-color     | #333                          | set the badge border color              |

<p>You can use a gradient or image for the background and text color.</p>
