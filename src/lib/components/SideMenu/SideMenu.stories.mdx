# Customize Side Menu - CSS Variables

You can customize the menu style using the following css variables.

## Colors

| Variable                             | Default                      | Description                                                              |
| ------------------------------------ | ---------------------------- | ------------------------------------------------------------------------ |
| --uai-nav-background-color           | #f0f0f0                      | The bg color of the menu, and button animation.                          |
| --uai-nav-txt-color                  | --uai-dark-txt               | The color of the menu itens and icons                                    |
| --uai-nav-items-hover-color          | --uai-light-surface          | The bg color of the menu itens                                           |
| --uai-nav-items-active-background    | --uai-dark-surface           | The bg color of the current active item                                  |
| --uai-nav-items-active-txt-color     | --uai-txt-on-dark-surface    | The text color of the current active item                                |
| --uai-nav-sub-items-active-highlight | --uai-items-active-txt-color | The underline color of the current active subitem, inside the group item |
| --uai-nav-spacer-color               | --uai-dark-txt               | The dash between the head and body and between the body and bottom       |

## Other variables

| Variable                            | Default                                                 | Description                                                                               |
| ----------------------------------- | ------------------------------------------------------- | ----------------------------------------------------------------------------------------- |
| --uai-nav-items-active-border       | none                                                    | You can set the border of the current active item instead of setting the background color |
| --uai-nav-animation-speed           | 350ms                                                   | Change the expand animation speed                                                         |
| --uai-nav-z-index                   | 9999                                                    | Set nav bar z-index                                                                       |
| --uai-nav-shadow                    | --uai-high-shadow                                       | Set the box shadow                                                                        |
| --uai-nav-border-radius             | 0 var(--uai-medium-shape) var(--uai-medium-shape) 0     | Set the nav border radius                                                                 |
| --uai-nav-items-border-radius       | 0px var(--uai-small-shape) var(--uai-small-shape) 0px   | Set the border radius of nav's items                                                      |
| --uai-nav-group-hover-border-radius | 0px var(--uai-medium-shape) var(--uai-medium-shape) 0px | Set the radius of the group item on hover                                                 |
